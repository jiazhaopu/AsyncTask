# Revolver 左轮手枪
## 分布式任务调度工具
### 简介
#### 轻量级、及时产生的分布式任务调度工具
1. 场景：
由于某个原因，产生了一个任务
- 单次任务。某个时间发一封邮件
— 重试机制。分布式事务数据一致性
- 重复调度任务

2. 支持任务类型：
- FIXED_TIME
    只能执行一次，需要指定执行时间，或者 CRON
- RETRY
    需要指定重试次数，如果指定 CRON 则按照规则重试，否则随机间隔 1～5 秒调度
- CRON
    按照 CRON 调度，可不指定次数上限

3. 支持分片、故障转移、秒级调度

4. 秒级别并发度计算：
任务调度线程池数(executePoolsNum) * 核心线程数(corePoolSize) / (1s / one task cost time)
